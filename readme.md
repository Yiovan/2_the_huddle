# Introduccion 


Una clase en python o ```class``` es una plantilla o molde para crear objetos 


## Que es la Programacion Orientada a Objetos 


Es un paradigma o estilo de programacion que organiza el código alrededor de "objetos" en lugar de funciones y lógica pura.


## Que es un class?

es como un molde o plano arquitectonico, es conocido como una plantilla que define datos y que comportamientos tendra un objeto 


#### Ejemplo:


un auto

- la clase seria el diseño de ingenieria, que dice todo los autos deben tener un motor, un color y la capacidad de acelerar

- El objeto seria el auto real que tienes estando afuera
